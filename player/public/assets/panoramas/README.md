# Panoramas Directory

Place your equirectangular panorama images here.

## Requirements

- **Format**: JPG, PNG, or WebP
- **Aspect Ratio**: 2:1 (width should be 2x height)
- **Resolution**: 4096x2048 recommended (2048-8192 width)
- **File size**: Under 10MB per image

## File Naming

The export system from the editor will use node IDs as filenames:
- `node-abc123.jpg`
- `node-def456.jpg`

## Testing

For local testing, you can use any equirectangular panorama image.
Free samples available at:
- https://polyhaven.com/hdris
- https://www.flickr.com/groups/equirectangular/

Download, rename to `sample.jpg`, and place in this directory.
